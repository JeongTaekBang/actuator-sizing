# Actuator Torque Tables

이 문서는 액추에이터 시리즈별 토크 테이블을 정리한 참조 문서입니다.
DB 구조 설계 및 실제 데이터 입력 시 참조용으로 사용됩니다.

---

## 시리즈 개요

| 시리즈 | 타입 | 복합 키 | 토크/추력 범위 | 전원 특성 |
|--------|------|---------|----------------|-----------|
| MA | Multi-turn | Model + kW + Hz + RPM | 40~15,680 Nm | 3상 전용 |
| MS | Multi-turn | Model + Phase + Hz + RPM | 45~110 Nm | 1상/3상 |
| NA | Part-turn | Model + Hz | 60~3,500 Nm | DC/AC |
| SA | Part-turn | Model + Type + Hz | 30~90 Nm | 1상 전용 |
| SR | Part-turn (Spring Return) | Model + Hz | 50~500 Nm | AC/DC |
| NL | Linear | Model + Hz | 4~35 kN | AC/DC |

## SR 시리즈 (Part-turn, Spring Return)

### 특징
- **복합 키**: Model × Hz
- **모델**: SR05, SR10, SR20, SR30, SR50 (5개)
- **Spring Return 기능**: Close-on-Fail (스프링에 의한 비상 폐쇄)
- **비대칭 동작시간**: Open = 모터 구동, Close = 스프링 구동 (1~2초)
- 사이징 시 Open 시간(모터 구동) 기준으로 선정

## NL 시리즈 (Linear)

### 특징
- **복합 키**: Model × Hz
- **모델**: NL04, NL08, NL20, NL35 (4개)
- **Linear 액추에이터**: 토크 대신 **추력(Thrust, kN)** 사용
- **기어박스 미사용**: 직접 연결만 지원
- **동작시간 계산**: `Stroke_mm / Speed_mm_sec`

### Multi-turn/Part-turn과의 차이

| 항목 | Multi-turn | Part-turn | Linear |

## DB 구조 설계 시 고려사항

### 복합 키 정리

| 시리즈 | Primary Key 구성 |
|--------|------------------|
| MA | Model + MotorPower_kW + Freq + RPM |
| MS | Model + Phase + Freq + RPM |
| NA | Model + Freq |
| SA | Model + Type + Freq |
| SR | Model + Freq |
| NL | Model + Freq |

### 공통 컬럼
- Model, Series, ActType, Torque_Nm, Freq, Weight_kg, BasePrice

### 시리즈별 추가 컬럼
- **MA**: MotorPower_kW, RPM
- **MS**: Phase, RPM, DutyCycle_min
- **NA**: OpTime_sec, DutyCycle_pct, MaxStemDim_mm (전압별 전류는 DB_ElectricalData)
- **SA**: Type, OpTime_sec, DutyCycle_min, MotorPower_W
- **SR**: OpTime_sec, MaxStemDim_mm (Spring Return - Close는 스프링 구동)
- **NL**: Thrust_kN, Speed_mm_sec, Stroke_mm (토크 대신 추력, 기어박스 미사용)

### 정규화 vs 비정규화 트레이드오프
- MA는 행 수가 많아짐 (Model × kW × Hz × RPM 조합)
- NA/SA는 상대적으로 단순
- VBA 검색 로직 복잡도 증가 고려 필요
